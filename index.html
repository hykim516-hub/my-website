Firebase 실시간 동기화 설정

PC와 모바일 전 기기에서 실시간 동기화하려면 Firebase 설정이 필요합니다.

설정 방법 (5분 소요)

  1. console.firebase.google.com 에서 새 프로젝트 생성
  2. Firestore Database 생성 (테스트 모드 선택)
  3. Authentication > 익명(Anonymous) 로그인 활성화
  4. 프로젝트 설정 > 웹 앱 추가 > firebaseConfig 복사
  5. 아래 입력창에 JSON 붙여넣기 후 저장
  6. 모든 기기에서 동일한 HTML 파일 사용

* 한 번 저장하면 이 기기에서는 새로고침해도 자동 연결됩니다.
